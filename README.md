# DreamMC Discord Bot

Бот, созданный для проекта [DreamMC](https://dreammc.su).

Список команд - `/help`

## Установка

Выполните:
```commandline
python -m pip install -r requirements.txt
```

Затем поместите в папку `DMC_Bot` файл `.env` со следующим содержание:
```.env
TOKEN="YOUR_BOT_TOKEN"
```

## Запуск

Выполните:
```commandline
python DMC_Bot/bot.py
```
